# Scenario Backgrounds Pending Real Art

The scenarios below currently ship with solid-color placeholder PNGs (1672x941) and WebPs. The runtime renders them with the standard background fallback path. Replace each `background.png` with curated art at the same dimensions and re-export to WebP under `public/assets/scenarios/{id}/background.webp`.
